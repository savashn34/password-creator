Create A Password



This is a personal password creator. It's for creating complex passwords that can be easily remembered. Password creation support is limited to 12-32 characters.

You must use only letters. No numbers, no marks. It will add them for you. You should just type something basic that you'll never forget. You can pick a keyword.


Here is an example usage:

Your keyword: mykeyword

Platform to use: github

Password: mykeywordforgithub
